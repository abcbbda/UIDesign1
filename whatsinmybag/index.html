What's in Yoonji's bag

Pocket1

이번 학기 편집디자인 수업을 들으면서 구입하게 된 클리어 파일이다. 거의 매주 인쇄물을 챙겨가야해서 아예 클리어 파일을 가방에 넣고 가져다닌다. 수업에서 진행한 과제물들과 근로장학생 출근부가 들어있다.


This is a clear file that I bought while taking this semester's editorial design class. I have to take the printout almost every week, so I carry the clear file in my bag. It contains the assignments conducted in the class and the attendance for working scholarship students.

Pocket2

20살 때부터 사용하고 있는 나의 아이패드이다. 가방에 맥북과 아이패드를 함께 넣으면 너무 무거워서 보통 집에 두고 다니는데, 오늘은 아이패드를 사용해야 하는 과제가 있어서 챙겨왔다. 보통은 과제보다 드라마나 영화, 예능을 보는 데 더 많이 사용한다.


This is my iPad that I have been using since I was 20. If I put my MacBook and iPad together in my bag, it's too heavy, so I usually leave it at home. But today I have a task to use my iPad, so I brought it. Usually, it is used more to watch dramas, movies, and entertainment than assignments.

작년부터 사용하고 있는 마우스이다. 내 맥북은 c타입 연결밖에 없어서 어쩔 수 없이 블루투스 마우스를 구매했는데, 생각보다 편하다. 손목이 편하다고 해서 버티컬 마우스를 샀는데 아직까지 손목이 괜찮은 걸 보니 잘 구매한 것 같다.


It is a mouse that I have been using since last year. My Macbook only has a c-type adapter so I had to buy a Bluetooth mouse, but it's more comfortable than I thought. I bought a vertical mouse because I heard it make a wrist comfortable. Seeing that my wrist is still okay, I think I bought it well.

올해 어머니께서 여행 갔다 와서 선물해주신 파우치이다. 기억은 잘 안 나지만 한 작가의 작품이 담긴 파우치라고 하셨던 것 같다. 파우치 안에는 립밤과 톤업크림, 헤어롤이 들어있다.


This is a pouch that my mother gave me after trip this year. I don't remember well, but I think she said it was a pouch designed by an artist's work. There are lip balm, tone-up cream, and hair rolls in the pouch.

Pocket3

어릴 때부터 공기놀이를 좋아해서 지금도 가끔 하는데, 작년에 경주여행을 갔을 때 구매한 공기놀이이다. 사실 당시에 가방에 넣어두고 귀찮아서 아직까지도 안 빼고 있다. 가끔 심심할 때 (가능한 상황이라면) 꺼내서 공기놀이를 하면 재밌다.


I like to play Gong-gi(Korean traditional game) since I was young, so I still play it sometimes, and I bought it when I went on a trip to Gyeongju last year. Actually, I put it in my bag at the time and I'm still lazy to take it out. Sometimes it's fun to take it out (if it is possible to play) and play Gonggi when you're bored.

작년 생일에 친구에게 선물받은 핸드크림이다. 여름이지만 가끔 건조할 때나 손을 씻고 나서 사용한다. 상큼한 과일향이라서 마음에 든다.


It's a hand cream that I got as a birthday gift from my friend last year. It's summer, but I sometimes use it when my hand is dry or after washing my hands. I like it because it smells like fresh fruit.
